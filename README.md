# Vrinda-Store-Data-Analysis-
Developed a comprehensive sales data analysis project for Vrinda Store using Microsoft Excel to derive actionable business insights and support data-driven decision-making. The project focused on analyzing large volumes of retail transaction data to understand customer behavior, sales performance, and product trends.
